# RAG service

## Setup Instructions

### 1. Create a Virtual Environment

```bash
python -m venv venv
```


### 2. Activate the Virtual Environment

```
source venv/bin/activate
```

### 3. Install Required Packages

```
pip install -r requirements.txt
```


### 4. Configuration

Store your OpenAI API key and LLM URL in config.py


### 5. Run the Server
```
python main.py
```
